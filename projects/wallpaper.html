4 Unique Wallpapers In One App

Explore diverse animated environments, from minimal interactive lines to a massive living village, all in one lightweight app.

1. LineFlow

Interactive Touch Reactive

A minimalist and elegant wallpaper featuring dynamic lines that instantly react and flow according to your touch gestures.

2. ParticleClock

Clock Face Particles & Butterflies

A mesmerizing clock display where glowing particles follow your fingertips, accompanied by beautifully animated butterflies.

3. MountainVillage

Weather System 3D Parallax

Our largest wallpaper! Features a dynamic sky system, moon phases, active weather (rain, snow, fog), 3-layered parallax mountains, a flowing river, and a lively village with houses, schools, and moving characters.

4. Background Studio

Custom Maker Gallery & Export

Create custom static wallpapers using solid colors, gradients, or custom images. Add noise, blur, or dim effects. Save to a local gallery, share them, or set them as custom backgrounds for LineFlow and ParticleClock!

Core Technologies & Systems

The modern Android development stack and advanced logic systems powering this application.

Kotlin & Canvas API

Custom rendering engine built entirely from scratch for ultimate control over every pixel and frame.

Real-Time Weather API

Fetches live weather data to dynamically trigger rain, snow, or fog within the MountainVillage environment.

Time & Celestial Sync

Synchronizes with device time to accurately reflect day/night cycles and calculate real-world moon phases.

Multi-Layer Parallax

Implements a 3-layered depth system calculating gyroscope and touch data for a stunning 3D visual illusion.

Seasonal Algorithms

Calculates current calendar dates to alter village aesthetics seamlessly across Spring, Summer, Autumn, and Winter.

Local File I/O & Storage

Utilizes secure local storage to save, retrieve, and manage custom image backgrounds created in the Studio.

Smart Battery Saver

Automatically pauses all rendering loops when the screen is off or app is hidden to ensure zero unnecessary battery drain.

Gesture Physics Engine

Advanced touch tracking for highly responsive fluid lines, interactive glowing particles, and swipe navigation.
