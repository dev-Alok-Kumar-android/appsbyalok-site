Echo Hunter

A fast-paced, lore-driven 2D cyberpunk survival game. Play as Protocol "Probe-7" and purge a corrupted mainframe. Built entirely from scratch natively in Kotlin without any external game engines.

Join Closed Beta View on GitHub

Core Systems & Gameplay

An engineering challenge: building a buttery-smooth cyberpunk experience with an ultra-small footprint using purely native APIs.

120FPS Custom Engine

Zero GC Canvas API

Engineered for a locked 120FPS on high-end devices. Uses pre-allocated primitive arrays (Object Pooling) to ensure absolutely zero garbage collection stutters during gameplay.

Cyberpunk Visuals

CRT Scanlines RGB Glitch

Features a 3D Parallax background grid, Overclock lightning trails, and Chromatic Aberration (RGB splits) on damage, delivering an authentic retro-hacker aesthetic without loading a single image sprite.

Hit-Stop Combat

Slow-Mo Heavy Impact

Ram enemies using the 'Overclock' mode. The custom collision system features God-of-War style Hit-Stop heavy impacts, shockwaves, and intense Slow-Motion Kill Cams for boss takedowns.

Meta-Narrative System

VHS Tearing Corrupted UI

A living game state. If you die during a glitch, the main menu permanently corrupts into a sliced "SYSTEM CORRUPTED" VHS-tearing effect across sessions. Features an epic "Interactive Core Merging" ending.

Three Distinct Modes

Story Endless Escape

Built via the Strategy Pattern, play 'Endless Waves', 'Story: Sector Cleanup' (featuring mutating bosses that fade in Hard Mode), and 'Firewall Breach' (a high-speed treadmill escape).

Micro-Footprint Audio

ToneGenerator Haptics

Heavy MP3 files were discarded. The game uses Android's hardware ToneGenerator to synthesize 8-bit sounds on the fly, paired with native haptic feedback for a tactile combat experience.

Core Technologies & Architecture

The native Android stack powering this lightweight application's engine.

Kotlin & Strategy Pattern

Written cleanly using the Strategy Pattern to decouple Game Modes, leveraging inline functions to keep performance strict.

Android Canvas API

No game engines used. Everything from the Parallax grid to glowing UI text is drawn directly on the Canvas using optimized Paint objects.

Persistent Memory Hub

A deeply integrated SaveManager handles async disk I/O to store unlocked endings, difficulty settings, and persistent UI corruption.

Zero Dependencies

Built targeting SDK 36 without importing external libraries for physics, rendering, or UI. Pure Android native development practice.
