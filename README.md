# Apps By Alok Website

Official portfolio website and privacy policy hosting for my Android applications published on Google Play.

## Live Website
https://appsbyalok.netlify.app

## About
This repository contains the source code for my personal developer website where I showcase:

- Published Android applications
- Open-source projects
- Technical skills
- App-specific privacy policy pages

## Project Structure
- `index.html`: Main portfolio website
- `style.css`: Stylesheet for the website
- `script.js`: JavaScript for interactive elements
- `privacy/`: Directory containing privacy policy pages for each app
